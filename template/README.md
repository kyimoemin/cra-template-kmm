# CRA Custom Template (without antd)

## used libraries

1. axios
2. react-query
3. redux
4. react router
5. tailwindcss

## dev dependencies

1. env-cmd

## Note

- tailwind preflight is disabled by default to enable antd style of antd components. you can reenable by `preflight: true` or comment it out in `tailwind.config.js`
